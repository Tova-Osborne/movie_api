MovieSearch API

Movie search REST API is built with node.js and express. The API returns key data in a JSON file pertaining to movies including movie description, genre, director, images (via URL).

Allows existing users to deregister https://[soontobenamedapp].com/users/{id} /users/:id DELETE Response: 200 "user {id} has been deleted." "user 1 has been deleted."
Allows users to add a movie to their list of favorites https://[soontobenamedapp].com/users/{id}/{movie_title} /users/:id/:movieTitle POST Response: 200 "{movie_title} has been added to the user {id}'s favorites list" Godfather has been added to the user 3's favorite list
Allows users to remove a movie from their list of favorites https://[soontobenamedapp].com/users/{id}/{movie_title} /users/:id/:movieTitle DELETE Response: 200 "{movie_title} has been removed from the user {id}'s favorites list" Godfather has been removed from the user 3's favorite list
